### Sobre

Desenvolvedor Full Stack e co-fundador da [Arbon](https://arbon-os.vercel.app), startup brasileira de HRtech que usa evidências comportamentais para tornar contratações mais precisas e justas.

No dia a dia, cuido do backend em NestJS/TypeScript, pipeline de IA, integrações com ATS e segurança da API — desde testes de penetração com OWASP ZAP até checklist de hardening para NestJS. Paralelamente, estou no 3º semestre de Técnico em Informática no CEAP e construo projetos de portfólio para solidificar a base técnica.

- 🔭 Trabalhando em: **Arbon** — pipeline de IA para hiring + integrações com ATS
- 🛡️ Estudando: **segurança de APIs** — pentest, BOLA/IDOR, OWASP Top 10
- 🌱 Aprofundando: **arquitetura de sistemas** e **infraestrutura**
